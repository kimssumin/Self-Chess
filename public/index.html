Game Rules
✔ 명령어 모음
 
A2->A3  |  A2 에서 A3 으로 이동
?A2  |  A2가 이동가능한 좌표를 보여줌
init {key} {value}  |  board 가 초기화된 상태에서, error 가 난 경우 수동적으로 key를 배치
set {key} {position} {color}  |  color의 key를 원하는 position 에 둠 (빈칸일 경우에만 놓을 수 있음)
display  |   board를 출력
possible {position}  |  position에 위치한 key의 이동가능한 경우를 모두 출력
✔ 실행규칙
 
1  백말부터 시작합니다
2  순서가 아닌 말을 이동하려 하면 경고창이 뜹니다
3  일반적인 체스 룰과 다르게 이동할 수 있는 경로에 말이 있을 때 상대 말을 제거하고 점수를 획득합니다
4  일반적인 체스와 다르게 킹의 역할이 없습니다 (체크메이트, 체크... X)
5  핀은 1점, 룩은 5점, 비숍과 나이트는 3점, 퀸은 9점으로 점수를 산정하는데,
현재 보드판에 놓인 체스말들의 점수 합으로 전체 점수를 계산합니다
